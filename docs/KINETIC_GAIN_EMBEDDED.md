## What this repo demonstrates

This repo demonstrates the **booking recovery control room** primitive — disruption queues tied to recovery blockers, guest communication posture, and operator-safe intervention lanes.

## How Kinetic Gain Embedded extends it

Kinetic Gain Embedded is security-first embedded analytics for B2B SaaS.
It extends this primitive to recovery-aware and measurement-aware in-product analytics for hospitality, travel, and service operations, wrapped in:

- signed JWT authentication
- semantic-layer row-level security (RLS)
- read-only data connectors
- AST SQL validation
- append-only audit events
